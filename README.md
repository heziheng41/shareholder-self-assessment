# 股东出资自测互动版

## 部署到 GitHub Pages

### 步骤1：创建仓库

1. 登录 GitHub：https://github.com/
2. 点击右上角 **"New"** 创建新仓库
3. 仓库名称：`shareholder-self-assessment`（或任意名称）
4. 勾选 **"Initialize this repository with a README"**
5. 点击 **"Create repository"**

### 步骤2：上传文件

1. 在仓库页面点击 **"Add file" > "Upload files"**
2. 拖拽以下文件到上传区域：
   - `股东出资自测互动版.html`
   - `README.md`（本文件）
3. 点击 **"Commit changes"**

### 步骤3：开启 Pages 服务

1. 进入仓库 **"Settings"**（设置）
2. 点击左侧菜单 **"Pages"**
3. 在 **"Source"** 部分：
   - 选择 **"main"** 分支
   - 选择 **"/ (root)"** 目录
4. 点击 **"Save"**

### 步骤4：获取访问链接

- 等待约1-2分钟，GitHub会生成链接
- 链接格式：`https://用户名.github.io/仓库名/股东出资自测互动版.html`

## 使用方法

- 将链接放入公众号文章的"阅读原文"
- 生成二维码图片插入文章正文
